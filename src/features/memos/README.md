# Memos 说说页面 for Flare Stack Blog

将 Memos 说说集成到 Flare Stack Blog 的独立页面。

## 功能

- ✅ 独立 `/memos` 说说页面
- ✅ 5 分钟缓存
- ✅ 与默认主题风格一致
- ✅ 支持置顶、标签

## 快速开始

### 1. 复制文件

将 `src/features/memos/` 目录复制到你的 Flare Stack Blog 项目中。

### 2. 添加路由

复制 `src/routes/_public/memos.tsx` 到你的项目。

### 3. 配置

修改 `src/features/memos/config.ts` 中的 `apiBaseUrl` 为你的 Memos Workers 地址。
